# 创建文章
   在hexo下创建一个新的文章
    ```
    heox new "文章名称"
    ```
  生产后会提示你文件路径，一般在hexo/source/_posts下
  文章基本设置
#创建标签
   创建标签页面
    ```
    hexo new page tags
    ```
#创建分类页面
    ```
    hexo new page categories
    ```

#打包页面 
    ```
       hexo generate
    ```

#启动本地服务 
    ```
        hexo server
    ```
